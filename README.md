# Points Boost

## Backend

### Setup

Setup virtualenv
```
    $ sudo easy_install virtualenv
    $ mkvirtualenv pointsboost
    $ cd backend
    $ pip install -r requirements.txt
```

Run server

```
$ python backend/app/routes.py 
```

## Frontend

* Install NodeJS 4.4.1
* `cd` in to the `frontend/` directory
* `npm install -g bower gulp`
* `npm install`
* `bower install`
* `gulp serve`
